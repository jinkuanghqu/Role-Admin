# Role-Admin
基于thinkPHP3.2开发的后台权限管理，此后台进行严格的权限控制以及更加细致化的控制,此后台仅仅是一个权限功能，可以直接使用，可根据业务功能进行相应的开发，为PHP后台开发人员大大提高了此步骤的开发过程，只需要开发人员在项目中添加其他功能开发即可。
